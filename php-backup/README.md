GoDaddy PHP form backup
=======================

These files are kept only as a backup if the site ever needs to move back to GoDaddy/cPanel hosting with PHP form handling.

Current live plan: Netlify hosting with Netlify Forms.

Backup files:
- `submit-form.php`: PHP form handler for cPanel hosting.
- `godaddy-root.htaccess`: clean URL rewrite rules for GoDaddy/Apache.
- `form-submissions.htaccess`: blocks direct browser access to saved CSV submissions.
